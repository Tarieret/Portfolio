StyleSignal: Trend Lifecycle Clustering

Not every trend behaves the same way over time — this project classifies trend behavior by shape rather than raw volume.

PythonpytrendsPandasScikit-learnUnsupervised LearningClusteringFeature Engineering
The Question

Does the shape of a trend's lifecycle — not just how big it gets — predict how long it stays commercially relevant?

The Approach

Search-interest time series for style and brand keywords are pulled via pytrends, then decomposed into shape-based features rather than clustered on raw interest values directly.

  • Extracting shape features: peak timing, rise rate, decay rate, duration above threshold, amplitude
  • Applying unsupervised clustering (K-means / hierarchical) to group trends into lifecycle patterns
  • Target patterns include spike-and-fade, sustained growth, and cyclical/seasonal return
  • Testing whether lifecycle shape predicts how long a trend remains commercially relevant
